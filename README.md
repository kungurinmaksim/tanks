# Tank Battle Game

Аркадная игра на Python с использованием Tkinter, где танки сражаются на процедурно генерируемой карте. Игрок управляет танком, а боты-танки перемещаются и стреляют.

## Особенности

- 🎮 **Управление игроком** - управление танком с помощью клавиатуры
- 🤖 **ИИ противников** - боты движутся к цели и случайным образом меняют направление
- 💥 **Система боя** - возможность стрелять и уничтожать вражеские танки
- 🗺️ **Различные типы местности**:
  - Обычная земля - нормальная скорость
  - Вода - снижение скорости в 2 раза
- 📦 **Боеприпасы** - подбор патронов на карте
- 🎨 **Разные спрайты** - различные текстуры для игрока и ботов

## Установка и запуск

### Требования
- Python 3.6 или выше
- Tkinter (обычно входит в стандартную установку Python)

### Запуск

cd tanks1/12_1_2

python main.py

### Управление

Клавиша	Действие
↑ (Стрелка вверх)	Движение вперед
↓ (Стрелка вниз)	Движение назад
← (Стрелка влево)	Движение влево
→ (Стрелка вправо)	Движение вправо
Пробел	Стрельба

### Основные классы

## Unit
Базовый класс для всех объектов на карте:

Движение и анимация

Обработка коллизий с картой

Система hitbox

## Tank
Класс танка (наследуется от Unit):

Управление игроком или ИИ

Боезапас (80 снарядов, макс. 100)

Подбор боеприпасов

Изменение скорости на разных типах местности

Система прицеливания для ботов

## Missile
Класс снаряда:

Быстрое движение

Автоматическое удаление при попадании

Принадлежность владельцу (для определения убийств)

Система коллизий
Столкновение с картой - объекты не могут покидать границы карты

Столкновение с объектами - танки не могут проходить друг через друга

Столкновение снарядов - попадание по танку наносит урон

### Специальные клетки:

Вода - снижение скорости

Боеприпасы - пополнение патронов

### Особенности ИИ ботов

Боты имеют два режима поведения:

Преследование цели (с вероятностью 90%) - движение в сторону игрока

Случайное движение (10% случаев) - изменение направления

Каждый бот с вероятностью 1/30 меняет поведение на каждом кадре.

### Настройка параметров

В коде можно изменить следующие параметры:

Tank._ammo - начальное количество патронов (по умолчанию 80)

Tank._speed - скорость танка (по умолчанию 2)

Missile._speed - скорость снаряда (по умолчанию 6)

Частота стрельбы ботов в missiles_collection.py
